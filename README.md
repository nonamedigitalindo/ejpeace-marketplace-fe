## Cara Initiate Project

git clone dan pull seperti biasa sudah works ya gaes, ga perlu command ribet lagi.

### Cara menjalankan aplikasi setelah dipull atau invite

**1. Install Packages**

```bash
npm install atau npm i
```

**2. Run dev all project**

```bash
npm run dev
```
